# Project4

This is a small Python practice project with simple scripts and a spammer
finder problem solution.

## Files

- `hello.py`: prints `hello world`
- `test1.py`: prints the result of `2 + 3`
- `spammer_finder.py`: finds the spammer account, runs test cases, and shows
  example outputs

## Run

Use Python from the project root:

```bash
python hello.py
python test1.py
python spammer_finder.py
```

## Virtual Environment

Create and activate a local virtual environment if needed:

```bash
python -m venv .venv
```

Windows PowerShell activation:

```powershell
.venv\Scripts\Activate.ps1
```
